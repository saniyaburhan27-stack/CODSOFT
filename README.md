# CODSOFT
About:
This project is a basic calculator web application developed using HTML, CSS, and JavaScript.
It can perform simple arithmetic operations such as addition, subtraction, multiplication, and division.
The goal of this project is to practice front-end web development concepts and improve logical thinking.

Features:
User-friendly interface
Performs basic arithmetic operations
Responsive design
Fast and accurate calculations
Beginner-friendly project

Technologies Used:
HTML – Structure of the calculator
CSS – Styling and layout
JavaScript – Calculation logic
